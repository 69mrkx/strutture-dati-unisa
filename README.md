# Progetto Didattico: Strutture Dati (PSD)

Benvenuti in questo repository, una collezione completa di implementazioni di strutture dati realizzate durante il corso di **Programmazione e Strutture Dati (PSD)** presso l'Università di Salerno (UNISA).

## Obiettivo
Il progetto nasce con l'intento di fornire agli studenti un riferimento pratico e commentato per comprendere e implementare le principali strutture dati studiate durante il corso, facilitando la preparazione per l'esame.

## Contenuto del Repository
All'interno troverai le implementazioni in **C**, sviluppate seguendo le linee guida del corso e ottimizzate per la chiarezza didattica. Le strutture incluse spaziano dalle primitive fondamentali fino a strutture più complesse:

- **Strutture Lineari:** Liste concatenate, Pile (Stack), Code (Queue).
- **Strutture Gerarchiche:** Alberi Binari di Ricerca (BST), Heap.
- **Strutture Avanzate:** Tabelle Hash.

## Configurazione e Utilizzo
Ogni cartella contiene il codice sorgente necessario. 

- **C:** Il progetto è strutturato per essere compilato su sistemi Unix con Makefile (Potete chiaramente creare un altro Makefile per il vostro sistema)

## Contributi e Supporto
Questo materiale è condiviso a scopo puramente didattico. Se sei uno studente del corso e hai suggerimenti per migliorare la leggibilità del codice o correggere implementazioni, sentiti libero di aprire una *Pull Request*.
